# 小红书直播数据分析方法论

> 从后台截图到诊断报告——完整的数据分析框架

---

## 四维诊断框架

### 维度 1：基础数据总览

| 指标 | 健康基准 |
|------|---------|
| 观看人数 | <500 偏低；500-2000 中等；2000+ 优秀 |
| 成交金额 | 按品类差异，首场有转化即合格 |
| 成交人数 | > 观看人数的 1% 正常 |
| 客单价 | 总成交 ÷ 成交人数，评估定价策略 |
| 平均观看时长 | <2min 差；2-5min 一般；5min+ 优秀 |
| 互动人数 | 低 = 缺少互动引导 |
| 新增粉丝 | 转化率 = 新增 ÷ 观看，>3% 优秀 |
| 预约人数 | =0 说明未做预热，最大流量缺口 |
| 退款金额 | 低 = 产品靠谱 |
| 观看-成交率 | >1% 正常 |

### 维度 2：观众画像

| 维度 | 分析要点 |
|------|---------|
| 性别 | 珠宝/美妆类 93%+ 女性为正常 |
| 年龄 | 主力年龄段决定话术风格（25-30 偏年轻 / 31-35 偏品质） |
| 消费力 | >50% 高消费力 = 中高客单价可行 |
| 城市等级 | 一二线主导 = 受众有购买力 |
| 人群标签 | 宝妈/白领/学生，不同标签不同打法 |

### 维度 3：流量来源结构

- **推荐流占比** — 自然推荐比例
- **笔记引流** — 0 = 直播前未发预告笔记
- **预约进入** — 0 = 未做预热
- **回访观众** — 低 = 没有固定开播节奏

**诊断逻辑**：
- 推荐流 > 80% 但总观看低 → 平台给了流量但没留住人
- 笔记引流 = 0 → 最常见问题，没做播前预热
- 回访 = 0 → 没固定开播时间，用户没养成习惯

### 维度 4：漏斗分析

```
曝光 → 进入 → 互动 → 商品点击 → 成交
```

| 环节 | 健康基准 | 偏低信号 | 优秀信号 |
|------|---------|---------|---------|
| 互动率 | ≥10% | <5% → 平台判定内容差 → 减少推流 | >15% → 增加推流 |
| 关注率 | ≥5% | 内容/人设弱 | 有吸引力 |
| 商品点击率 | ≥8% | 话术未引导 | 展示有效 |
| 转化率 | ≥2% | 价格/信任问题 | 商品+话术到位 |
| 平均观看时长 | ≥2min | 内容留不住人 | 有可看性 |

## 流量竞争分析

| 推荐流占比 | 诊断 |
|-----------|------|
| >70% | 平台还在给流量，问题是留存/互动 |
| 40-70% | 推流在减弱，检查互动率 |
| 20-40% | 推流明显减少，检查有无违规/水印问题 |
| <20% | 大直播间抢流量，换时段或加预算 |

---

## 新手扶持期

新账号有 **3-7 场临时流量扶持**：
- 第 4 场开始流量下降 ≠ 账号出问题，是扶持结束
- 此时要从「吃扶持」切换为「可持续运营」

---

## 互动率 — 核心指标

| 互动率 | 平台反应 |
|--------|---------|
| <5% | 判定内容无趣 → 减少推流 |
| 5-10% | 正常推流 |
| >10% | 增加推流 |

**提升互动 4 招**：
1. 开场 3 分钟不卖货，先提问：「姐妹们你们戴银饰会过敏吗？」
2. 福袋：「扣 1 抽小礼物」，设「关注 + 评论」条件
3. 每 5 分钟：「刚进来的宝宝扣个 1」「想要这款的扣想要」
4. 目标：互动率 **>10%**

## 常见问题速查

- **投薯条后数据反而差？** — 可能人群不匹配。笔记内容和直播间卖的不一致会污染数据。停投，恢复 2-3 场自然流量后重新校准。
- **高流量后突然暴跌？** — 大概率是新手扶持结束。解法：固定开播节奏 + 预告笔记 + 每场上新。
- **高客单价反而好卖？** — 说明观众消费力强，不要降价，走品质路线。

## 输出结构

每次诊断按以下结构输出：

```
## 总览
（一句话概括本场直播表现）

## 逐条数据解读
- 观看 253 人 — 偏低但新账号可接受
- 互动 12 人，互动率 4.7% — 低于 5% 阈值
- 推荐流 73% — 平台还在推，不是被抢流量

## 交叉验证
- 互动率 4.7% + 推荐流 73% → 不是缺流量，是缺留存
- 平均观看 2 分 34 秒 + 预约=0 → 开场+预热都有问题

## 推理逻辑
- If 推荐流高但总观看低 → 平台给了初始推流但没留住人
- If 互动率<5% → 平台判定内容差 → 减少推流
- If 预约=0 → 没有预热笔记 → 流量只靠推荐流一条腿

## 对照案例/行业参考
- 珠宝类新账号首月互动率中位数：8-12%

## 优化建议（5 段结构）
### 流量篇 — 播前笔记、标签、开场 30 分钟
### 留人篇 — 钩子频率、福利品、互动话术
### 转化篇 — 主推品话术、组合销售、逼单
### 节奏建议 — 分时段脚本表（0-10min, 10-30min...）
### 一句话总结
```
