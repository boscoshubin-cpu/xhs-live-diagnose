# 🚀 小红书直播数据诊断工具

> 专为国内直播运营打造——上传截图，秒出诊断报告。适配普通运营 + AI Agent 双场景。

![screenshot](screenshot.png)

## 💡 为什么做这个

小红书直播运营每天下播第一件事：打开后台，盯着一堆数字——观看多少？成交多少？转化率掉了没？流量从哪来的？

数据都在后台摆着，但要自己**手动算转化率、拆漏斗、对比趋势**，一场直播复盘动不动半小时起步。很多中小直播间根本没专人做数据分析，主播下播已累瘫，哪有精力再拉 Excel。

所以做了这个——**上传一张直播后台截图，自动出诊断报告**。不用算、不用写、不用学。

> 让直播运营省下复盘时间，去做更有价值的事。

---

## 🎯 三种使用方式

### 🖥 方式一：普通运营（双击即用）

1. 下载 `index.html`，双击用浏览器打开
2. 拖入直播数据后台截图（PNG/JPG/HEIC）
3. 对照截图填写数据 → 点「开始诊断」
4. 出报告：5 个核心指标卡 + 四维诊断 + 优化建议

> ⚠️ 纯前端，数据不出浏览器 🔒

### 🤖 方式二：Hermes Agent（装 Skill）

```bash
hermes skills install boscoshubin-cpu/xhs-live-diagnose
```

装完 Agent 就能帮你分析直播数据——发截图直接出诊断报告，附带互动话术模板和分时段脚本。

### 🤖 方式三：Cursor / Cline / Claude Code / Copilot 等

把 `AGENTS.md` 的内容复制到你的 Custom Instructions / `.cursorrules` / `.clinerules`：

- 内置完整诊断知识库（健康基准、漏斗阈值、观众画像标准）
- 固定的输出结构模板（总览 → 解读 → 交叉验证 → 推理 → 建议）
- 用户发直播截图，Agent 直接按框架输出诊断报告

|------|---------|
| 📊 基础数据 | 观看 / 成交 / 转化率 / 客单价 / 停留时长 |

| 🏷️ 价格结构 | 福利款 / 入门款 / 利润款 / 高端款 |
